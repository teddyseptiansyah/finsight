## KONTEKS PROYEK

Aplikasi Vite + React (single-page)
Tujuan Akuntan upload Excel export Navision → dashboard laporan keuangan otomatis
Cara kerja sistem baca nomor akun COA dari kolom A, SUMIF ke akun-akun yang sudah dipetakan

Cara jalankan
  cd finsight
  npm install
  npm run dev
  # build npm run build → dist

## ARSITEKTUR PENTING

### App.jsx — 2 file upload terpisah
- File LR (wajib) net change per bulan → parseSheet()
- File Neraca (opsional) balance at date → parseBalanceSheet()
- `handleFiles(lrPayload, nrcPayload)` → set data → Dashboard

### Dashboard.jsx — data flow
- `filteredLr`      → TabLabRugi, TabArusKas (lrData=), RatioPanel, KpiDrilldown, VarianceSection
- `filteredNeraca`  → TabNeraca, TabPiutang, TabArusKas (neracaData=), RatioPanel
- `hasNeraca`       → disable tab Neraca jika file neraca tidak diupload
- CoaContext.Provider membungkus semua → value { openCoa, lrData filteredLr }

### Tab data props
  TabLabRugi   data={filteredLr}    months={activeMonths}
  TabNeraca    data={filteredNeraca} months={activeMonths}
  TabArusKas   lrData={filteredLr}  neracaData={filteredNeraca} months={activeMonths}
  TabPiutang   data={filteredNeraca} months={activeMonths}
  VarianceSection filteredLr={filteredLr} ... (lihat Dashboard.jsx)

### CoaContext — WAJIB import dari lib, bukan dari Dashboard
  import { CoaContext } from '....libCoaContext'   ← ui dan tabs
  import { CoaContext } from '..libCoaContext'       ← Dashboard.jsx

## TAB-SPESIFIK

### TabNeraca
- Filter akun prefix 1x2x3x, bukan akun LR (4x-9x)
- Buang SUMMARY_NOS (xxxxxs999999), nama mengandung TOTALi
- Dropdown per grup Aktiva Lancar, Aktiva Tetap, Aktiva Lain, Hutang Lancar, Hutang JK Panjang, Ekuitas
- Tiap grup expand → mini tren chart + tabel sub-akun
- Grafik 2 pie (aktiva & pasiva), line tren aktiva vs pasiva, stacked bar aktiva, multi-line komponen

### TabArusKas
- Baca dari neracaData (bukan lrData)
- Akun Kas  prefix 1101x — HANYA by nomor akun (NO fallback by nama)
- Akun Bank prefix 1102x — HANYA by nomor akun (NO fallback by nama)
- Arus kas = delta saldo (saldo[i] - saldo[i-1])
- Tabel saldo dropdown — header KasBank klik untuk expand detail akun

### TabPiutang
- Baca dari neracaData
- Filter nama akun mengandung piutangi (bukan by prefix)
- Buang TOTAL rows, no. akun ending 99999999999

### TabLabRugi
- zigzag S-flow design
- sign helper getProfitColor(v), getExpenseColor(v)
